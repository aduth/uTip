µTip Demo

Basic

Useful for when you simply want a tooltip with some static text, and you're fine with the default styling.

Hover here

new uTip(document.getElementById("example1"), { text: "Hello world!" });
$("#example1").uTip({ text: "Hello world!" });

Dynamic text

Similar to the basic example, but allows you to recalculate the tooltip text on every mouseover.

Hover here or place your cursor over me

new uTip([ document.getElementById("example2"), document.getElementById("example3") ], {
    text: function(context) { return "You're hovering <strong>" + context.innerHTML + "</strong>"; }
});
$("#example2,#example3").uTip({
    text: function(context) { return "You're hovering <strong>" + $(context).text() + "</strong>"; }
});
Note: As you see here, µTip can use the same tooltip for multiple elements. Simply pass an array of elements if you're using plain JavaScript, or a jQuery object of multiple elements if you're using jQuery.
Note: Because the text callback function is passed the mouseover event, you can manipulate the return value based upon the target element.

Colors

One of the simplest customizations you can make is to add some color. Simply provide a base background color and µTip will find complementing colors for the background gradient, border, and text.

Color me RED

Color me YELLOW

Color me BLUE

new uTip(document.getElementById("example4"), {
    backgroundColor: "#ff3f3f",
    text: "Hello RED!"
});
new uTip(document.getElementById("example5"), {
    backgroundColor: "#fffbcf",
    text: "Hello YELLOW!"
});
new uTip(document.getElementById("example6"), {
    backgroundColor: "#7cd1ff",
    text: "Hello BLUE!"
});
$("#example4").uTip({ backgroundColor: "#ff3f3f", text: "Hello RED!" });
$("#example5").uTip({ backgroundColor: "#fffbcf", text: "Hello YELLOW!" });
$("#example6").uTip({ backgroundColor: "#7cd1ff", text: "Hello BLUE!" });
Note: Currently, backgroundColor only supports hex values (e.g. "#000" or "#ff3f3f"), not color names or RGB(A) values (e.g. "Green" or "rgba(0, 0, 0, 0.25)")
Note: When the background color is perceived to be "dark", the text will display in a lighter color (refer to the red tooltip in this example). See the following example for how to manipulate this further (specifically, the "darkThreshold" and "textColor" options).

All options

There are a variety of options for manipulating the display of the tooltip. The following example, while hideous, makes use of every available option.

Hover here

new uTip(document.getElementById("example7"), {
    backgroundColor: "#ccffff", // See example: "colors"
    borderBrighten: -0.8,       // Percentage of pure white to add to border color
    gradientBrighten: -0.5,     // Percentage of pure white to add to gradient target
    textColor: "#ff9b66",       // Overide calculated text color
    textShadow: false,          // Toggle text shadow display (CSS3 only)
    darkThreshold: 0.1,         // At which perceived brightness to start using light color, as percentage of pure white
    maxWidth: "150px",          // Corresponds to CSS max-width value
    padding: "2px",             // Corresponds to CSS padding value
    transitionTime: "0.2s",     // Fade in time, corresponding to CSS transition-duration value (CSS3 only)
    transitionDelay: "0.5s",    // Fade in delay, corresponding to CSS transition-delay value (CSS3 only)
    text: function() {
        return "Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore " +
            "et dolore magna aliqua.";
    }
});
$("#example7").uTip({
    backgroundColor: "#ccffff", // See example: "colors"
    borderBrighten: -0.8,       // Percentage of pure white to add to border color
    gradientBrighten: -0.5,     // Percentage of pure white to add to gradient target
    textColor: "#ff9b66",       // Overide calculated text color
    textShadow: false,          // Toggle text shadow display (CSS3 only)
    darkThreshold: 0.1,         // At which perceived brightness to start using light color, as percentage of pure white
    maxWidth: "150px",          // Corresponds to CSS max-width value
    padding: "2px",             // Corresponds to CSS padding value
    transitionTime: "0.2s",     // Fade in time, corresponding to CSS transition-duration value (CSS3 only)
    transitionDelay: "0.5s",    // Fade in delay, corresponding to CSS transition-delay value (CSS3 only)
    text: function() {
        return "Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore " +
            "et dolore magna aliqua.";
    }
});
Note: As a reference, here are the default options:
  • backgroundColor: "#eee"
  • borderBrighten: -0.2
  • gradientBrighten: 0.05
  • textColor: undefined
  • textShadow: true
  • darkThreshold: 0.5
  • maxWidth: "300px"
  • padding: "8px"
  • transitionTime: "0.2s"
  • transitionDelay: "0s"
  • text: function() { return ""; }
